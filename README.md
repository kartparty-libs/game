# 🏁 KartParty  
*Web3 Kart Racing — Fast, Fun, and Fully Owned*

KartParty is a casual, blockchain-native racing game where speed meets sovereignty. Race colorful NFT karts across dynamic tracks, earn rewards through skill—not grind—and own every asset on-chain.

## ✨ Features  
- **Play Instantly**: No wallet needed to start—embedded onboarding for all players.  
- **Own Your Ride**: Collect, upgrade, and trade karts as verifiable NFTs (Ethereum L2).  
- **Race & Earn**: Win $PARTY tokens in daily races, tournaments, and challenges.  
- **Govern & Shape**: Vote on new tracks, balance changes, and features with $PARTY.  
- **Built for Everyone**: Web + mobile support, social lobbies, and intuitive controls.

## 🚀 Get Started  
1. Visit [kartparty.net](https://www.kartparty.net/)  
2. Create your racer (no crypto knowledge required)  
3. Race your first lap in <30 seconds
